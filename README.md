### 🚀 GitHub Pages 部署目录

这是由 [Nuxt 4 Blog](https://github.com/yingzya/yingzya.github.io) 自动部署生成的静态站点内容。

### 📄 说明

- 本分支用于 GitHub Pages 托管静态网页
- 所有文件均由 GitHub Actions 自动生成于 `.output/public`
- 不建议手动修改本分支内容

### 🔄 自动更新方式

每次 `main` 分支代码更新并推送后，GitHub Actions 会：

1. 构建最新的 Nuxt 静态站点；
2. 将生成的 `.output/public` 内容自动推送到本分支；
3. GitHub Pages 自动发布更新内容。

查看部署页面 👉 [MyBlog](https://yingzya.top/)
